## 🚀 Quick Start

### Requirements
- WordPress 6.0+
- WooCommerce 8.0+
- PHP 8.0+
- An LLM API key (or local Ollama instance)

### Installation

```bash
# Option 1: Composer
composer require vimes1984/woo-agentic-checkout

# Option 2: Manual
cd wp-content/plugins/
git clone git@github.com:vimes1984/woo-agentic-checkout.git
cd woo-agentic-checkout
composer install --no-dev
```

Activate from **Plugins → Woo Agentic Checkout**. Configure your LLM provider in **WooCommerce → Agentic Checkout 🍌**.

### First Run

1. Set your LLM API key in Settings
2. Choose a self-healing permission level (start with **Suggest**)
3. Wait for agents to run on their cron schedules (or click "Run Now")
4. Review suggestions in the Suggestions tab
5. Approve confidently, escalate to Auto-Patch
